## 项目介绍

​		本项目为一个内存调页模拟程序。在本项目所模拟的系统中，执行程序共有320条指令。一页可以存放10条指令，所有指令被分为32页，系统中共有4个页框可供程序使用。本项目将展示程序执行过程中的换页过程，以及缺页率的统计。



## 运行方法

```bash
git clone https://github.com/ZhichenRen/Memory-Management.git
cd Memory-Management
yarn install
yarn serve
```
